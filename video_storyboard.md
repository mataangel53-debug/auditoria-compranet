# Guion y Storyboard: Auditoría COATLICUE - El Caso Mata vs. Grupo Salinas

**Formato:** Video Animado (Motion Graphics)
**Estilo:** Futurista, minimalista, legal. Paleta de colores dark/light mode (negro/verde/blanco o blanco/verde/negro).
**Música de fondo:** Sutil, electrónica, con un tono de urgencia controlada y revelación.
**Duración estimada:** 2 - 3 minutos.

---

### **ESCENA 1: EL PROBLEMA - LA FRAGILIDAD DE LA EVIDENCIA DIGITAL**

**(0-15 segundos)**

| NARRADOR (Voz en off, calma, autoritativa)                                                                                             | VISUALES                                                                                                                                                                                                                                                          |
| -------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| "En la era digital, un documento puede ser alterado con un clic. Una fecha, una firma, una palabra... cambiada sin dejar rastro."       | Pantalla negra. Aparece un documento digital flotando en el centro. El texto es nítido. Un cursor de mouse aparece, selecciona una cifra monetaria y la cambia. El documento original se desvanece y el alterado queda, indistinguible.                                     |
| "¿Cómo podemos confiar en la justicia si la evidencia misma es frágil? ¿Cómo puede un ciudadano defenderse si sus pruebas pueden ser invalidadas?" | El documento alterado se multiplica en miles de copias idénticas, llenando la pantalla, creando una sensación de caos e incertidumbre. La palabra "INCERTIDUMBRE" aparece y se disuelve.                                                                              |

| "...por despido injustificado contra un poderoso conglomerado. ### **ESCENA 5: LA PRUEBA - EL ÁRBOL DE LA VERDAD**

**(90-120 segundos)**

| NARRADOR                                                                                                                            | VISUALES                                                                                                                                                                                                                                                          |
| ----------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| "Tenemos una contradicción. ¿Cómo sabemos que estos documentos son reales? ¿Que no fueron alterados? Con matemáticas."                | Los bloques de hash de cada documento (demanda, rechazo, solicitud, etc.) se alinean en la parte inferior de la pantalla.                                                                                                                                           |
| "El Protocolo COATLICUE toma la huella digital de cada página de nuestro informe de auditoría..."                                    | Vemos el informe HTML (`informe_final_coatlicue.html`) en pantalla. Se divide visiblemente en páginas. De cada página sale su propio hash.                                                                                                                      |
| "...y las combina en pares, creando nuevos hashes. Este proceso se repite, construyendo un Árbol de Merkle."                          | Animación clara y simple del Árbol de Merkle. Dos hashes se unen, una línea sube y un nuevo hash se forma. El proceso se repite con todos los hashes de las páginas, subiendo de nivel, hasta que solo queda un único hash en la cima.                          |
| "Este es el Root Hash. Una sola huella digital que sella la integridad de todo el informe. Si un solo pixel cambia, este código cambiará." | El Root Hash final (`<merkle_root_del_informe>`) brilla en la cima del árbol. Se muestra grande en el centro de la pantalla.                                                                                                                                          |
